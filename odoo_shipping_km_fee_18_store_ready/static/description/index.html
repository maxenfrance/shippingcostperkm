Distance-Based Shipping Fee

Automatically calculate delivery cost based on KM and time

This module uses the Google Maps Distance Matrix API to calculate the shipping fee for each order based on the customer's distance from your warehouse.

Features

  • Calculate distance and travel time via Google Maps API
  • Add cost per kilometer and per hour to Sales Orders or Invoices
  • Manual entry wizard if customer collects order themselves
  • Configure prices and API key via Settings
